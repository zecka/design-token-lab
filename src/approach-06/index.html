Approach 06

Stateful Contrast Pairs

Each role exposes variant-ready contrast pairs for rest and interaction states. Components choose a variant recipe while the theme owns every surface, ink, border, and focus color.

Token Architecture

Facet Surface role Intent
--pair-[role]-highlight-bg vivid fill Strong role expression for high-emphasis badges.
--pair-[role]-highlight-fg text on vivid fill Contrast-matched label ink for the vivid surface.
--pair-[role]-highlight-border vivid border Edge color aligned to the highlight surface.
--pair-[role]-highlight-bg-hover vivid hover fill Theme-tuned hover continuation of the highlight pair.
--pair-[role]-highlight-bg-active vivid pressed fill Pressed high-emphasis surface that preserves contrast.
--pair-[role]-soft-bg faint background Quiet tinted role surface for low-emphasis badges.
--pair-[role]-soft-fg text on faint background Readable role-colored ink for soft surfaces.
--pair-[role]-soft-border faint border Subtle edge for separating soft surfaces.
--pair-[role]-soft-bg-hover faint hover background Slightly stronger soft surface on hover.
--pair-[role]-soft-bg-active faint pressed background Pressed state for quiet controls.
--pair-[role]-outline-bg ambient background Resting outline background, often transparent.
--pair-[role]-outline-fg outline text Readable role ink for outline badges.
--pair-[role]-outline-border outline stroke Visible role stroke for outline controls.
--pair-[role]-outline-bg-hover outline hover wash Tinted wash introduced by outline hover.
--pair-[role]-outline-border-hover outline hover stroke Stronger outline edge for hover and active emphasis.
--pair-[role]-focus-ring focus indicator Accessible role-associated focus ring.
--pair-[role]-focus-gap focus separation Local surface gap that keeps focus visible.
--pair-[role]-disabled-bg disabled background Muted inactive surface assigned through the pair contract.
--pair-[role]-disabled-fg disabled text Muted inactive text without relying on opacity.
--pair-[role]-disabled-border disabled border Muted inactive edge that keeps shape stable.
--surface-canvas page background Theme foundation behind components.
--surface-panel raised surface Panel and demo cell background.
--surface-panel-alt nested surface Secondary surface for labels and nested content.
--surface-border structural border Neutral edge for demo frames and dividers.
--surface-border-hover structural hover border Theme-level hover edge for later surface components.
--ink-strong primary text Main readable text on neutral surfaces.
--ink-muted secondary text Supporting readable text on neutral surfaces.
--ink-subtle metadata text Lowest-emphasis theme text.
--component-bg resolved component background Local alias selected from a role pair by variant and state.
--component-fg resolved component text Local alias selected from the paired foreground token.
--component-border resolved component edge Local alias selected from the pair border token.

--component-focus-gap resolved focus gap Local alias for focus separation.
