Approach 02

Facet Contract Tokens

Roles publish a shared set of color facets, then badge variants map those facets into component slots. This keeps naming compact while making every role behave predictably across solid, soft, and outline treatments.

Token Architecture

--role-primary-fill: #9a6a2e → --badge-fill: var(--role-primary-fill) → background: var(--badge-fill)
--role-success-tint: #e7eee5 → --badge-tint: var(--role-success-tint) → background: var(--badge-tint)
--role-evening-line-hover: #5e527d → --badge-line-hover: var(--role-evening-line-hover) → border-color: var(--badge-line-hover)
